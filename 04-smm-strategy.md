## Воронка продаж: 5 уровней

### Уровень 1 — AWARENESS (Осведомлённость)

**Инструменты:**
- TikTok Reels: ASMR литья, истории храмов, «как рождается украшение»
- Instagram Reels: алгоритмный охват новой аудитории
- Таргет ВКонтакте: интересы — путешествия, архитектура, история, искусство
- Коллаборации: 5–10 микроинфлюенсеров (архитекторы, тревел-блогеры, историки)
- PR: статьи в нишевых медиа

**Контент-крючки:**
```
🔥 «Этот узор мы нашли на стене монастыря 12 века»
🔥 ASMR: расплавленная бронза заливается в форму
🔥 «5 вещей, которые объединяют все храмы мира»
🔥 «Чем украшения из бронзы лучше, чем из золота?»
🔥 Сравнение: масс-маркет VS украшение с историей
```

**KPI:** 50K+ просмотров на TikTok, 20K+ на Reels

---

### Уровень 2 — INTEREST (Интерес)

**Что делаем:**
- Идеально выстроенный профиль: Highlights, лента, описание
- Stories каждый день: живые, человечные, закулисные
- Лид-магнит: PDF «История 7 великих храмов мира» → в Telegram

**Контент для INTEREST:**
```
📖 Карусель: «История Храма Покрова на Нерли + украшение»
📖 Видео: «Что означает этот узор?» — рассказывает основатель
📖 Stories: «Я была в Армении — вот что я привезла оттуда»
📖 Живой эфир: «Задай вопрос ювелиру»
📖 Тест в Stories: «Какое украшение твоё по знаку зодиака?»
```

**Welcome-серия e-mail:** 5 писем о бренде, вдохновении, мастерах

---

### Уровень 3 — DESIRE (Желание)

**Инструменты:**
| Инструмент | Механика |
|-----------|---------|
| Отзывы и UGC | Реальные покупатели в Stories и постах — каждый день |
| Распаковки | Видео: как выглядит упаковка, первое впечатление |
| Гарантии | Пост: «Мы гарантируем качество. Вот почему.» |
| Лимитированность | «Осталось 3 украшения из коллекции Нерль» |
| Ретаргет | Реклама на тех, кто заходил на сайт, но не купил |
| E-mail nurturing | «Вы смотрели это украшение — оно всё ещё ждёт вас» |

**Триггеры покупки:**
```
⏰ «Коллекция ограничена — следующего литья не будет 3 месяца»
💌 «Только для подписчиков: -10% по промокоду TEMPLE»
🎁 «Купи до 15-го — получи подарочную упаковку и открытку»
👥 «87 человек добавили это в избранное»
⭐ «Катя из Москвы носит уже 2 года и не снимает»
```

---

### Уровень 4 — ACTION (Покупка)

**Оптимизация точки продажи:**
- Сайт: чистая навигация + история каждого изделия
- Быстрый ответ в Direct (< 2 часов) — скрипты для менеджера
- WOW-упаковка: коробочка с историей храма-вдохновения внутри
- QR-код на открытке → аудио или видео от основателя

**Скрипт ответа в Direct:**
```
«Привет! Это украшение из нашей коллекции [название] —
вдохновлено [храм/место]. Мы делаем его вручную методом
литья по выплавляемым моделям. Материал: бронза с покрытием 999 золотом.

Хочешь узнать историю этого изделия? Я пришлю тебе маленький
рассказ о месте, которое его вдохновило 🌍»
```

---

### Уровень 5 — LOYALTY (Преданность)

**Программа лояльности:**
```
🥉 1-я покупка → благодарственное письмо + маленький бонус
🥈 2-я покупка → доступ в закрытый Telegram-клуб «Хранители»
🥇 3+ покупки → персональный промокод + ранний доступ к коллекциям
👑 Амбассадор → бесплатное изделие за пост с тегом бренда
```

**UGC-кампания:**
```
📸 Хэштег: #sevenworlds_story
📸 Механика: «Покажи своё украшение и расскажи —
   куда ты бы взял его в путешествие?»
📸 Лучший пост — репост в аккаунте бренда + подарок
```

## Стратегия с инфлюенсерами (KOL)

**НЕ работает:**
- Звёзды с 1М+ (дорого, нецелевой охват)

**РАБОТАЕТ — микроинфлюенсеры 10–100К в нишах:**
- 🏛️ Архитектура и история
- 🌍 Путешествия (travel-блогеры)
- 🎨 Искусство и дизайн интерьеров
- 👗 Осознанная мода (slow fashion)
- 💆 Духовность / медитация / сакральное

**Формат работы:** сначала бартер, затем — небольшой гонорар за лучших

- Подписчики: +3000–5000

- CPL (цена лида): < 300 руб.
- ROAS: 3–5x

### Месяц 6 (масштаб)
- Подписчики: 10K+
- Амбассадоры: 10–15 активных
- Медиа-упоминания: 5+
- Выручка через digital: ×2 к старту
