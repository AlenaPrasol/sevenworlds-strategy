# 📅 Контент-план Sevenworlds

## Формула контента: 4-3-2-1

```
4 поста — Образование (история, культура, материалы)
3 поста — Вдохновение (атмосфера, эстетика, архитектура)
2 поста — Продажа (конкретный товар, отзыв, оффер)
1 пост  — Личное (основатель, команда, за кулисами)
```

## Матрица форматов

| Формат | Частота | Цель | Пример |
|--------|---------|------|--------|
| **Reels** | 3×/нед | Охват | ASMR литья / История храма |
| **Карусель** | 2×/нед | Вовлечение | «7 фактов о бронзе» |
| **Статичное фото** | 2×/нед | Эстетика | Украшение + архитектура |
| **Stories** | Ежедневно | Доверие | Опросы, закулисье, отзывы |
| **Стрим** | 2×/мес | Конверсия | Эфир с ювелиром / основателем |
| **TikTok** | 5×/нед | Вирусность | Трендовые звуки + украшения |

## Недельный ритм

| День | Формат | Тема |
|------|--------|------|
| **Пн** | Образовательный Reel | «Как рождается украшение: от эскиза до металла» |
| **Вт** | Stories-история | «Этот узор — с армянского монастыря 12 века» |
| **Ср** | Лайфстайл-фото | Украшение в руках / на теле в красивом интерьере |
| **Чт** | UGC / отзыв | Покупатель показывает своё украшение |
| **Пт** | Reel-«атмосфера» | Видео ручного литья, рабочий процесс |
| **Сб** | Коллаборация | Совместный пост с блогером или архитектором |
| **Вс** | Вдохновение | Фото храма + украшение, вдохновлённое им |

## ТОП-10 идей для Reels (высокий потенциал охвата)

1. 🔥 ASMR: расплавленная бронза заливается в форму (без слов, только звук)
2. 🏛️ «Этот узор — из монастыря 12 века» (было/стало: старинное фото → украшение)
3. 📦 Распаковка: «Вот как мы упаковываем украшение с историей»
4. ✋ Процесс ручного литья за 60 секунд
5. 🌍 «5 храмов, которые вдохновили наши украшения» (карусель в формате Reel)
6. 🎭 «Что люди думают, узнав цену — и что думают, узнав историю»
7. 📖 Основатель рассказывает: «Почему я основала бренд после ООН»
8. 🔬 «Почему бронза с 999 золотом — это честнее, чем позолоченное серебро»
9. 👗 Styling: «3 образа с одним украшением» (коллаборация с fashion-блогером)
10. 🕌 Путешествие: «Я в [Стамбул/Рим/Армения] — вот что я вижу глазами дизайнера»

## Идеи для Stories (ежедневно)

### Вовлечение:
- Опрос: «Какой храм мира тебе нравится больше?»
- Тест: «Угадай, какой регион вдохновил это украшение»
- Вопрос: «Куда бы ты взяла это украшение в путешествие?»

### Доверие:
- Отзыв дня (скриншот или фото покупателя)
- «За кулисами»: фото мастерской / процесса
- Личное: «Екатерина сегодня в...»

### Продажа:
- Товар дня с историей
- «Осталось N штук»
- Промокод для подписчиков Stories

## Идеи для Telegram-канала «Хранители миров»

- 📖 Еженедельная история одного места / храма
- 🎁 Эксклюзивные офферы только для подписчиков
- 🎙️ Голосовые сообщения от Екатерины из путешествий
- 📸 Фото «до публикации» — первыми видят новые украшения
- 🏛️ Рубрика «Артефакт недели» — объект искусства из мировых музеев

## Позиционирование контента

### Tone of Voice
```
✅ Умный, но не снобский
✅ Поэтичный, но не пафосный
✅ Образовательный, но не скучный
✅ Личный, но не фамильярный
```

### Визуальный стиль
```
🎨 Цвета: золото, охра, терракот, глубокий зелёный, бежевый камня
📸 Свет: естественный, тёплый, «музейный»
🏛️ Фоны: архитектура, камень, фактурные поверхности, льняные ткани
📐 Ракурс: крупный план деталей украшения, руки мастера
```
